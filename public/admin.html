Integrations

n8n Webhook

When set, all check-in notifications are sent here. n8n handles Slack DMs, #security posts, and Google Calendar. Takes priority over the direct Slack webhook.

Slack Bot Token

When set, visitors are announced by DMing the host directly. Requires a Slack User ID on the employee record. Takes priority over the channel webhook below.

Slack Channel Webhook

Fallback when no bot token is configured, or when the host has no Slack User ID. Posts to a channel.
